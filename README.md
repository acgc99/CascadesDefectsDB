# CascadesDefectsDB

This repository is a curated database of collision cascade defects from molecular dynamcis simulations whose main purpose of is to be used as input for [irradiapy](https://github.com/acgc99/irradiapy). Subfolders indicate the recoil energy in eV.

For irradiapy version 2.x, use the latest commit.

For irradiapy version 1.x, use the commit `494c3cdaca684d0874b582f69409573b3e3bcbdb` of this repository for compatibility.
